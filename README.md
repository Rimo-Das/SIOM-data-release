# SIOM-data-release
* Spin-Induced-Multipole Moment based tests provide us a way to investicate the BBH nature of a compact object. Here, SIOM refers to Spin-Induced-Octupole Moment. We test the BBH nature of GW241011 by our methodology and provide first stringent constrain on SIOM parameter.
* This project is a public version of siom-analysis inherited from [here](https://git.ligo.org/rimo.das/siom-analysis). You can only access this link only if you are a LIGO member. But this public repository is very much self explanatory.
* The aim of this project is to reporoduce the Figures from this [paper](https://arxiv.org/abs/2604.09828)
